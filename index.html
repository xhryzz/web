BSPWM Parrot OS

Este repositorio contiene la configuración y scripts para establecer un entorno de trabajo utilizando bspwm en Parrot OS. Incluye personalizaciones y herramientas útiles para mejorar la productividad.

Spotify Premium IOS

Este repositorio proporciona un cliente de Spotify con características premium, sin necesidad de una cuenta premium. Permite disfrutar de música de manera ilimitada y sin anuncios.

Python Spotify API Tkinter

Características:

  • Interfaz gráfica de usuario
  • Sin anuncios
  • Descargas de música
  • Listas Custom

Automated Task Scheduler

Este repositorio contiene un sistema de programación de tareas automatizado que permite a los usuarios programar y ejecutar tareas recurrentes en segundo plano. Ideal para automatizar procesos repetitivos en entornos de desarrollo y producción.

Python Cron Docker

Características:

  • Programación de tareas recurrentes
  • Ejecución en segundo plano
  • Integración con Docker
  • Registro de ejecuciones

XhyTrack

XhyTrack es una aplicación full-stack responsiva diseñada para ayudar a las personas a llevar un control de sus entrenamientos. Los usuarios pueden agregar ejercicios, ver estadísticas/historial de sus entrenamientos.

Express MongoDB Node.js

Características:

  • Agregar ejercicios a entrenamientos
  • Ver estadísticas
  • Consultar historial completo
  • Interfaz responsiva y fácil de usar
